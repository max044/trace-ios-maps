# trace-ios-maps

Packs cartographiques (MBTiles vectoriels, dérivés OSM/Protomaps) pour l'app **TRACÉ** (iOS, offline).
Public pour permettre le téléchargement sans authentification depuis l'app.

- `catalog.json` : liste des régions disponibles (nom, bbox, URL du pack, taille).
- Les packs `.mbtiles` sont hébergés dans les **Releases**.
